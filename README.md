## 🧠 Key Learnings
### Monotonic Stack
Used in **Daily Temperatures** and **Exclusive Time of Functions**. 
* **Logic:** Keeps a stack sorted to find the "next greater/smaller" element in $O(n)$ time.
* **Pro-tip:** Store the *index* instead of the *value* to calculate distances easily.
